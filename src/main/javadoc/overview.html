This is a command-line utility application for underwater photographers to automatically add EXIF tags to their image files with latitude, longitude, and/or altitude (depth) based on logged dive profiles and GPS tracks. It can read exported log files from a variety of common dive computers and GPS receivers. The main purpose was to assist with the Bay Area Underwater Explorers (BAUE) Geo-Tagging Project.

GPS receivers don't work underwater. So in order to record a GPS track the diver must either log the location before descending (or after surfacing), attach the receiver to a towed surface float, or put the receiver in a buoyant waterproof container and send it to the surface on a line during the dive.
